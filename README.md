Turpial Airlines es una simulación de sitio web para una aerolínea ficticia Venezolana, diseñada para demostrar la integración de funcionalidades clave en plataformas de e-commerce. La aplicación permite a los usuarios explorar vuelos, agregar opciones a un carrito de compra, gestionar favoritos y simular el proceso de pago, replicando la experiencia de una plataforma real. El enfoque del proyecto está en la implementación de lógica de interacción, manejo de estados y flujo de compra dentro de un entorno web dinámico. Este proyecto funciona como un entorno de prueba para validar estructuras de frontend y la integración de componentes esenciales en sistemas de reserva y compra online.
